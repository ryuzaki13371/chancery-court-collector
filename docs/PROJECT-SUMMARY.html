Lead Collector — Project Summary

Automated public-records lead collection for Hamilton County, TN

Prepared by M. Rizki Kurniawan  ·  Client: Steven R.  ·  June 2026

What this is

Three automations that collect leads from public Hamilton County government records, clean and organize them, and deliver them automatically — into a Google Sheet, to Telegram, and as ready-to-read PDF/CSV files. Everything runs in the cloud for free. No server, no monthly subscription.

🏠

Obituaries → Addresses

Obituary names from chattanoogan.com, reformatted to “Last First”, each searched on the County Trustee property site to find a property address.

📋

Court Docket Names

Individual party names pulled from the Chancery Court “Current Motion Call Docket” PDFs (Part 1 & 2).

💰

Tax Sale → Owners

Every property on the Delinquent Tax Sale list — address, parcel, minimum bid — with the current owner looked up on the Trustee site.

How it works

SOURCE (gov website / PDF) │ ▼ COLLECTOR (Python) ── scrapes & cleans the data │ ▼ GitHub Actions (cloud) ── runs it free, weekly + on demand │ ├──► Google Sheet ── auto-fills, deduplicated, one tab per task ├──► Telegram bot ── sends a neat PDF + a CSV └──► Run artifact ── downloadable copy on the run page

A small Telegram bot (@LeadCollectorHQ_bot) lets anyone pull the latest list on demand: open the bot, press Start, tap a button, and the file arrives in the chat in a minute or two. Pressing Start also subscribes that person to the automatic weekly delivery.

Why some address rows are blank (this is normal, not a fault). The Trustee site only lists the current owner. Someone who recently passed often isn’t the owner of record anymore — the home may be in a spouse’s name, a trust, or already sold. So you get solid addresses on many names and blanks on others. Every row is labelled (Matched / Possible — verify / No match) so it’s transparent.

How to use it (for Steven)

  1. The Google Sheet updates by itself — just open it. Three tabs: Chancery CourtObituariesTax Sale
  2. Pull a list any time from your phone: open t.me/LeadCollectorHQ_bot → press Start → tap a button. The file arrives in the chat.
  3. Weekly delivery arrives automatically once you’ve pressed Start. Send /stop to pause it, /start to resume.

Reading the columns

Obituaries

  • Search Name — the “Last First” version searched
  • Property Address — best match (blank if none)
  • Other Possible Addresses — extra candidates
  • Match — Matched / Possible — verify / No match / Common name — skipped

Tax Sale

  • Sale Status — Active (going to sale) vs Redeemed
  • Property Address · Minimum Bid
  • Owner (from Trustee) — the lead
  • Lookup — Owner found / Vacant land / Not found
